# Changesets

Este diretório gerencia versionamento e changelogs via
[changesets](https://github.com/changesets/changesets).

- `pnpm changeset` — cria um changeset descrevendo mudanças.
- `pnpm version-packages` — aplica os changesets (bump de versão + changelog).
- `pnpm release` — publica (packages não-private).
